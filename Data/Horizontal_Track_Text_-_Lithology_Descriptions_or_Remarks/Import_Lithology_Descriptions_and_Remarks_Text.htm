Import Lithology Descriptions and Remarks Text

Import text from another application (such as MS wordTM). 

It should have the following format.

 

Depth <tab> Text

<blank line>

Depth <tab> Text

 

  1. Select the text and copy to clipboard.

  2. In GEO, on the Tools tab, select Track Text Manager.
    The Track Text Manager dialog box will open.

  3. From the Track List, select the track that you want to import the text into.

  4. Click Import.

  5. A confirmation dialog box will open. Click Yes.

  6. Click Exit.

 

Note. The imported text will adopt the current settings (such as fonts, text margins, justifications and text type).
